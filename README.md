# Test App

## Setup

 - git clone [https://github.com/gargrahul925/test-app](https://github.com/gargrahul925/test-app)
 - cd test-app
 - docker-compose up

## Tech Stack

 - NodeJS
 - ExpressJS
 - MongoDB
 - Redis
 - Mongoose
 - Jest

## Notes
